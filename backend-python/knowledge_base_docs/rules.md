# Тема: Новый экран

Вопрос: Не работает новый экран?
Ответ: Проверьте интернет соединение на экарне, если интернет есть, попробуйте перезапустить KDS, закрыв его через крестик. Если не помогло, то перейдите в "Настройки" которые находятся слева снизу и нажмите кнопку "Обновить". Если не помогло обратитесь к Марии.

Вопрос: Не печатает экран?
Ответ: Перейдите в *"Настройки"* которые находятся слева снизу и нажмите кнопку *"Обновить"*. Проверьте  Если не помогло обратитесь к *Марии*.
